optimizationTools.geometryTools package#

geometryTools#

This package contains useful tools for modifying a FreeCAD document and handling import and export from python

optimizationTools.geometryTools.face_to_step(doc, face_name, output_path)[source]#

Export a FreeCAD FaceBinder object to a .step file

Parameters:

  • doc (Document) – FreeCAD document

  • face_name (str) – Label of the FaceBinder object that is exported

  • output_path (str) – path where the .step file is saved

optimizationTools.geometryTools.face_to_stl(doc, face_name, output_path, max_length=0.4)[source]#

Export a FreeCAD FaceBinder object to a .stl file using FreeCAD’s .stl exporter

Parameters:

  • doc (Document) – FreeCAD document

  • face_name (str) – Label of the FaceBinder object that is exported

  • output_path (str) – path where the .stl file is saved

  • max_length (float) – resolution of .stl mesh (max length of any triangle side in mm)

optimizationTools.geometryTools.merge_stls(stl_paths, output_path)[source]#

Merge multiple .stls into one preserving their distinction by utilizing stl regions

Parameters:

  • stl_paths (dict) – dictionary faceName -> stl_path

  • output_path (str) – output path of combined .stl

optimizationTools.geometryTools.modify_sketch(doc, sketch, datum_name, datum_val, unit='mm')[source]#

Set a datum (constraint) in a sketch to a value. The datum is found by its label

Parameters:

  • doc (Document) – FreeCAD document

  • sketch – sketch object (type: Sketcher.SketchObject; no type hinting possible)

  • datum_name (str) – name of the datum that is modified

  • datum_val (float) – new value for the specified datum

  • unit (str) – ‘mm’ for lengths or ‘deg’ for angles

optimizationTools.geometryTools.prepare_output_directory(path)[source]#

Create an output directory if it does not exist yet.

Parameters:

path (str) – path to directory that is checked. Should not be a path to file!

optimizationTools.geometryTools.prepare_output_path(path)[source]#

Check if the output file is to be saved in a directory and if true, prepare the output directory

Parameters:

path (str) – path to output file

Convert a .step file to .stl using gmsh

  • step_path (str) – path to .step file

  • stl_path (str) – path where the generated .stl file is saved

  • stl_resolution (float) – Resolution of the triangulated mesh in the .stl
